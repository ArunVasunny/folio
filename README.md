# Folio Platform

A portfolio hosting platform for photographers, interior designers, architects, and creatives.

## Project Structure

```
folio-platform/
├── frontend/          # Next.js 14 App Router
└── backend/           # Node.js + Express + MongoDB
```

## Quick Start

### Backend
```bash
cd backend
npm install
.env   # fill in your MONGODB_URI
npm run dev
```

### Frontend
```bash
cd frontend
npm install
.env.local   # fill in NEXT_PUBLIC_API_URL
npm run dev
```
